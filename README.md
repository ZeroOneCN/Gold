# Trading Analysis - 黄金白银交易分析平台

一个专注于 **XAUUSD（黄金）和 XAGUSD（白银）** 交易数据分析的桌面平台。支持交易记录管理、统计分析、保证金计算、AI 智能分析及出入金管理。

---

## 功能概览

### 统计分析
- 交易总览卡片（笔数、净盈亏、胜率、手续费、做多/做空分布）
- 账户资金联动（累计入金/出金、净入金、当前净值、收益率）
- 每日盈亏曲线 & 盈亏分布饼图
- 品种分析表（按 XAUUSD / XAGUSD 汇总）
- 盈亏比（平均盈利 ÷ |平均亏损|）

### 交易记录
- 完整的增删改查，支持品种和买卖方向筛选
- Excel / CSV 批量导入，自动去重
- 编辑时自动计算：手续费（按手数）、持仓时间（按时段）、盈亏（按买卖方向）
- 分页浏览（首页 / 末页 / 跳转页码）

### 交易计算器
- **多仓位共同计算**：支持同时添加多个仓位（多品种、多方向），统一汇总保证金与盈亏
- 每个仓位独立计算：合约价值、保证金、点值、预计盈亏
- 汇总结果：总保证金、总合约价值、净值、保证金比例、总盈亏
- 自动获取账户当前净值作为余额，杠杆默认 500
- 仓位数据自动缓存在本地，刷新不丢失

### AI 智能分析
- 选择日期范围，一键分析交易表现
- 从整体评价、问题/优势、品种偏好、风控建议四个维度输出结论
- 结果缓存 24 小时，刷新不丢失
- 需配置 DeepSeek API Key 后使用

### 资金管理
- 出入金记录（入金 / 出金），支持编辑和删除
- 与交易数据联动：净值 = 净入金 + 盈亏 + 手续费
- 收益率自动计算

## 快速开始

### 环境要求
- **Node.js** >= 22
- **npm** >= 10

### 安装与启动

```bash
# 1. 克隆仓库
git clone https://github.com/ZeroOneCN/Gold.git
cd Gold

# 2. 安装依赖
npm install
cd client && npm install && cd ..

# 3. 构建前端
npm run build

# 4. 启动服务（生产模式）
npm start
```

浏览器打开 **http://localhost:3001**

### 开发模式

```bash
# 终端 1：启动后端 API
npm run server

# 终端 2：启动前端热更新
npm run client
```

前端开发端口 **http://localhost:5173**（自动代理 API 到 3001）

## AI 分析配置（可选）

如需使用 AI 智能分析功能，需配置 DeepSeek API Key：

**文件配置：**
```bash
cp .env.example .env
vim .env
```

不配置 Key 不影响其他功能使用，点击分析时会提示「未配置 API Key」。

## 导入交易数据

1. 进入「交易记录」页面，点击「下载模板」获取标准 Excel 模板
2. 按模板格式填入交易数据
3. 拖拽文件到上传区域或点击上传
4. 系统自动去重（按日期+品种+类型+开仓价+手数+开仓时间判断），不会重复导入

---

## 注意事项

- 数据库文件 `trades.db` 首次启动自动创建，请勿手动删除
- XAUUSD 价格保留 2 位小数，XAGUSD 价格保留 3 位小数
- 盈亏金额、手续费、手数统一保留 2 位小数
